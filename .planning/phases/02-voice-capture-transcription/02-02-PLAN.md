    - Export `runtime = 'nodejs'`
    - Export `maxDuration = 60`
    - `POST(request: Request)` logic:
      1. Auth guard:
         - Use `createClient()` from `@/lib/supabase/server`
         - `supabase.auth.getUser()`; return 401 JSON if missing user
      2. Parse formData and read `audio` file
      3. Validate:
         - file exists
         - file size > 0
         - file size <= 25MB
      4. Call Whisper:
         - `client.audio.transcriptions.create({ model: 'whisper-1', file, prompt: SALES_WHISPER_PROMPT })`
      5. Return JSON:
         - `{ transcript, mimeType, sizeBytes }`
      6. Error handling:
         - 400 for bad input
         - 413 for oversized file
         - 502 for provider errors
         - 500 fallback with request-safe message

    Keep response body stable because 02-03 UI will consume this contract.
  </action>

      - missing file returns 400
